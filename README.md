# Dodge the Meteors

A polished browser-based 2D arcade game built with HTML, CSS, vanilla JavaScript, and HTML5 Canvas.

## How to Play

Open `index.html` in a browser and press Play.

- Move with A/D or the Arrow Keys.
- On mobile, use the Left and Right buttons.
- Avoid meteors for as long as possible.
- Collect blue shield orbs to block one hit.
- Collect purple slow-time orbs to slow meteors for 5 seconds.
- Pass close to meteors for a Near Miss bonus.

## Features

- Responsive canvas scaling for desktop and mobile
- Main menu, instructions modal, pause menu, and game-over screen
- Animated starfield, rotating meteors, ship flame, particles, screen shake, and floating score text
- Difficulty scaling over time
- High score saved with localStorage
- Generated sound effects, simple looping background music, and mute toggle
